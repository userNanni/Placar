# PocketBase + Fly.io

Este repositório contém os arquivos necessários para publicar seu servidor PocketBase online com Fly.io.

## Como usar:

1. Coloque o binário do `pocketbase` nesta pasta e renomeie para `pocketbase`.
2. Coloque sua pasta `pb_data/` com a coleção `placar` dentro da mesma pasta.
3. Execute os comandos:

    fly launch
    fly deploy

Seu servidor estará acessível em: https://simplyheron.fly.dev
